# Deyor — Tour Packages Website

A modern React.js tour packages website for [Deyor.in](https://www.deyor.in/) — multi day tours, bike trips, community adventures, and international getaways. Featured on Shark Tank India Season 2.

## Features

- **Homepage** with animated hero, stats counter, featured tours, collections, reviews, FAQ
- **18+ tour packages** from Deyor.in (Ladakh, Spiti, Bhutan, Thailand, Vietnam, and more)
- **10 tour collections** (Ladakh Bike, Maldives, Spiti, Kashmir, etc.)
- **GSAP animations** — scroll reveals, hero entrance, stats counter, review carousel
- **Tours listing** with search, category filters, domestic/international filters
- **Tour detail pages** with highlights, pricing, and WhatsApp booking
- Fully responsive design

## Tech Stack

- React 19 + Vite
- React Router DOM
- GSAP + ScrollTrigger + @gsap/react
- Lucide React (icons)
- Pure CSS

## Getting Started

```bash
npm install
npm run dev
npm run build
```
